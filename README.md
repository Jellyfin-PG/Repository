# Jellyfin Plugin Group Repository

A repository, for plugins created by this group

---

## Requirements

- Jellyfin 10.11+

All plugins are available from the repository below.

---

## Installation

**1. Add the plugin repository**

In Jellyfin, go to **Dashboard > Plugins > Repositories** and add:

```
https://raw.githubusercontent.com/Jellyfin-PG/Repository/refs/heads/main/manifest.json
```
